# Getting Started

Welcome to the sample project documentation. This wiki provides high-level software documentation for your projects.

## Features

- **Markdown Support** - Write documentation in Markdown
- **Mermaid Diagrams** - Create flowcharts, sequence diagrams, and more
- **Git Integration** - Link to your repository for source code
- **Project Organization** - Organize docs by project and category
